機能
- 同盟ログ検索機能:ログカテゴリタブ(「全て」除く)で検索画面が出現
- トレード(10TP即落札一覧、出品中カードへのダイレクトリンク、リスト一括入札)
- 半自動チュートリアル
- 自動ブショーダス(自動削除付)
- 自動ヨロズダス
- 武将図鑑未取得カードのトレードリンク
- トレード関連書簡自動開封削除
- クエスト補助
- 出兵予約時刻
